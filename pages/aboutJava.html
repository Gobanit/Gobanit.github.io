Java is an object-oriented programming language, that runs on Java Virtual Machine. That should make java applications run on any platform that supports JVM (in theory).
Practically it doesn't work just like that, but you can be almost sure that desktop version will work on all the basic desktop systems (Windows, Linux, Mac) without issues.

Still, you need to download JVM (if you haven't already) for running Java applications. Check the latest version or download it here .
If you are interested in programming in Java, you need also Java Development Kit(JDK), probably with some developer environment (see here).
